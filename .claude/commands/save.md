現在の作業内容をGitHubに保存してください。

手順:
1. `git status` と `git diff --stat` で変更を確認し、**今回のゴールに関係するファイルだけ**を明示的に `git add <path...>` する（`git add -A` は禁止＝身に覚えのない差分・巻き戻りを巻き込む事故を防ぐ）。無関係な差分があれば add せず原因を確認する
2. 変更内容を日本語でまとめたコミットメッセージを作成
3. git commit
4. git push origin claude/[現在のブランチ名]
5. 報告: 「保存しました。変更内容: [要約]」+ `git diff --stat` の1行サマリ
